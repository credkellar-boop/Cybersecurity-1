# Contributing to Cybersecurity-1

First off, thank you for considering contributing to **Cybersecurity-1**! 🛡️ 

This repository is built by the community, for the community. Whether you are fixing a typo, adding a new hands-on lab, or updating exam questions for the 2025 certification standards, your help is deeply appreciated.

---

## 📜 Code of Conduct
By participating in this project, you agree to maintain a respectful and welcoming environment for everyone. We do not tolerate gatekeeping or toxic behavior. We are all here to learn and secure the digital frontier together.

---

## 🛠️ How You Can Help
Here are a few ways you can contribute to the repository:
* **Add Labs:** Create detailed, step-by-step markdown guides for tools like Nmap, Wireshark, Splunk, or AWS IAM.
* **Update Syllabuses:** Keep the Phase 1, 2, and 3 checklists updated with the latest vendor exam objectives.
* **Expand the Testing Engine:** Add new Python dictionaries of questions to our quiz scripts in the `Testing_and_Exams` folder.
* **Fix Typos & Links:** Found a broken resource link or a spelling error? Submit a quick fix!

---
