# ☁️ Cloud Security Engineer Track

Welcome to the Cloud Security Engineer track. This path is focused on securing cloud infrastructures, protecting data across distributed environments, and mastering platform-specific security tools for AWS and Azure.

### 🗺️ Your Roadmap
* **Phase 1:** Security+ (Completed via `00_Foundation`)
* **Phase 2:** AWS Security Specialty / Azure Security Engineer
* **Phase 3:** CCSP (Certified Cloud Security Professional)

Navigate to the `01_AWS_Azure_Security` directory to begin Phase 2 of your journey!
