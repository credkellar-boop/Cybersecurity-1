# 🔎 Phase 2: CySA+ Foundations

This module focuses on the operational side of defense. You will learn how to leverage intelligence and threat detection techniques, analyze and interpret data, identify and address vulnerabilities, and effectively respond to and recover from incidents.

### 🎯 Objectives
By the end of this module, you will understand:
* Threat intelligence and threat hunting.
* Security Information and Event Management (SIEM) systems.
* Vulnerability management lifecycle.
* Digital forensics and incident response (DFIR) fundamentals.

---

### 📚 Study Checklist

#### 1. Threat and Vulnerability Management (22%)
* [ ] Read: The Vulnerability Management Lifecycle
* [ ] Lab: Utilizing CVSS (Common Vulnerability Scoring System)
* [ ] Lab: Configuring and running credentialed scans

#### 2. Software and Systems Security (18%)
* [ ] Read: Secure Software Development Life Cycle (SDLC)
* [ ] Lab: Analyzing web application logs for injection attacks

#### 3. Security Operations and Monitoring (25%)
* [ ] Read: Threat Intelligence Sources (STIX, TAXII, Open Source)
* [ ] Lab: Setting up and querying a SIEM (e.g., Splunk or ELK Stack)
* [ ] Lab: Creating custom IDS/IPS rules (e.g., Snort or Suricata)

#### 4. Incident Response (22%)
* [ ] Read: The NIST Incident Response Process
* [ ] Lab: Conducting basic memory and disk forensics
* [ ] Lab: Isolating and containing a compromised endpoint

#### 5. Compliance and Assessment (13%)
* [ ] Read: Data privacy and handling regulations
* [ ] Read: Frameworks for security maturity (e.g., CMMC)
